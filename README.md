# RetreivBE
RetreivBE is an optimized search engine backend for PostgreSQL, written in Typescript. 
It features basic search engine functions such as sorting, full-text-search, and filters,
as well as a highly configurable indexing rule, caching, built-in rate-limiting and an event-driven asynchronous indexing pipeline.

- [ ] Search index
